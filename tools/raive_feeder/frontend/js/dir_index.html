tools/raive_feeder/frontend/js/

Client-side JavaScript modules for raiveFeeder SPA. Each module is a self-contained IIFE attached to the window namespace.

Contents

NameTypeDescription
app.jsFileFeederApp — tab navigation, shared API helpers (apiFetch, apiUpload, formatBytes)
upload.jsFileFeederUpload — drag-and-drop document upload with metadata form and batch progress
audio.jsFileFeederAudio — audio upload, provider toggle (local/API), transcribe → edit → ingest flow
images.jsFileFeederImages — image upload, mode selector (flier/scan), preview grid, OCR → edit → ingest
scraper.jsFileFeederScraper — URL input, depth/pages sliders, NL query, crawl results with selective ingest
corpus.jsFileFeederCorpus — stats dashboard, source table with delete, semantic search, export/import
progress.jsFileFeederProgress — WebSocket connect/disconnect for real-time batch job progress
batch.jsFileFeederBatch — job list, cancel/pause/resume API wrappers
